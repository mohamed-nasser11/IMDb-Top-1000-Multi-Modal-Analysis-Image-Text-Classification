# IMDb-Top-1000-Multi-Modal-Analysis-Image-Text-Classification
This project builds a multi-modal machine learning pipeline using data scraped from IMDb across 4 genres: Action, Drama, Comedy, and Thriller. We collected 1000 unique movies including poster images, titles, release years, ratings, vote counts, and genre labels through deep automated web scraping using Playwright, bypassing dynamic JavaScript rendering and anti-bot mechanisms to extract real DOM data directly from the live IMDb website.
